# Rimuru – Leyenda Eterna

Juego narrativo interactivo de evolución y decisiones en un mundo isekai fusionado (Tensei Shitara Slime Datta Ken + Overlord + otros).

## Características

- Sistema de evolución del slime
- Estadísticas dinámicas (Poder, Sabiduría, Reputación, etc.)
- Narrativa ramificada
- Integración con IA (Groq recomendado)
- Guardado automático con localStorage
- Modo Survival y Mazmorra
